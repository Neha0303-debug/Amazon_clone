# 🛒 Amazon Clone (Frontend)

A static Amazon homepage clone built using HTML and CSS. This project focuses on replicating the layout and design of the Amazon website to improve frontend development skills.

---

## 🚀 Features

* Amazon-like homepage UI
* Navigation bar with logo, search bar, and menu
* Product sections and banners
* Responsive design for different screen sizes
* Clean and structured layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox & Grid)

---

## 📂 Project Structure

* index.html
* style.css
* images/

---

## 💡 What I Learned

* Building complex layouts using Flexbox and Grid
* Structuring large webpages using HTML
* Styling components to match real-world websites
* Improving UI design and responsiveness
